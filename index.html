Habilidades

Dataset

Consumer API

É essencial para integrar sistemas e transferir dados entre dispositivos de forma eficiente, tenho experiencia em rest e webservice. Principalmente em dados de geolocalização - GPS, já trabalhei com API da SASCAR - GPS (Satelital), SS Telematics - GSM (Dados Moveis), Autotrack - GPS (Satelital) com foco na visualização no OpenStreetMap, tudo voltado a telemetria do caminhão e sua localização em áreas de interesse, EX: Caminhão chegou no cliente, e-mail disparado ou Bot Telegram avisa etc.

Desenvolvedor de APIs

Tenho experiência no desenvolvimento de APIs com com base no Fast API. Focando principalmente em sistemas que exigem uma camada de segurança.

Bots

Automatizei tarefas manuais e repetitivas no computador para agilizar o fluxo de trabalho, como cadastros usando PyAutoGui, coleta de dados em sites usando Selenium, edição em massa de fotos usando OpenCV, coleta de dados no e-mail, e recentemente venho buscando novos conhecimentos em ChatBots com IA com base no Botpress e n8n.

Front-End

Não é o meu maior forte, mas sei lidar bem com HTML, CSS e JS para construção de sites ou formulários para Fluig, e hoje estudo o React-Js.

Back-End

Esta é minha principal área de atuação, com foco total em Python (Full-Stack). Trabalho com arquitetura de software, desenvolvimento de bancos de dados relacionais e microserviços, garantindo escalabilidade, eficiência e segurança nas aplicações que as implemento em Docker.

Visão Computacional

Inteligência Artificial - (IA)

Tenho conhecimento básico em implementação de modelos classificatórios e preditivos, utilizando desde algoritmos clássicos como MLP (Multi-Layer Perceptron), SVM (Support Vector Machines), Decision Tree, Random Forest, até modelos mais simples como Regressão Linear, Regressão Logística, KNN (K-Nearest Neighbors), K-Means e DBSCAN.

Power Bi

Tenho conhecimento básico em Power BI para criação de dashboards. Já desenvolvi um painel com KPIs logísticos, proporcionando aos gestores uma visualização clara e estratégica dos dados.

Virtual Machine (VM)

Versionamento de Código (GitHub|GitLab)
